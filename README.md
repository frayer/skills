# Agent Skills

A collection of [Agent Skills](https://agentskills.io/) for Claude Code and other agentic AI systems.

## Skills

### work-journal

A professional productivity advisor for senior technical leaders. Work Journal helps capture, organize, and reflect on work through three workflows:

1. **Weekly Tactical Prep** — Transform rough work notes into polished weekly updates aligned to your strategic objectives
2. **Performance Reviews** — Synthesize accumulated weekly notes into mid-year or year-end self-assessments
3. **Strategic Advisor** — A conversational mentor grounded in your objectives, principles, and recent work context

Work Journal operates as a persistent advisor that reads your knowledgebase (objectives, principles, and current efforts) to provide context-aware guidance — not generic corporate-speak.

### setup-work-journal

A one-time setup skill that scaffolds the complete directory structure and template files needed to use work-journal. Creates knowledgebase templates, directory layout, and onboarding documentation with placeholder content ready to customize.

## Installation

Install these skills using the `npx skills` command documented at [skills.sh](https://skills.sh/docs):

```bash
npx skills install work-journal
npx skills install setup-work-journal
```
